[Function]
get-lf w

w

Return: returns : LF_parent, agreement (in a variable list format, if more than one possibility) and part of speech

(get-lf 'leave) returns ((LF_LEAVE-TIME (AGR (? Y (1S 2S 1P 2P 3P))) V) (LF_LEAVE (AGR (? Y (1S 2S 1P 2P 3P))) V) (LF_DEPART (AGR (? Y (1S 2S 1P 2P 3P))) V)) Note that agreement features are (NIL) for words that don't have any (adjectives, adverbs...)
